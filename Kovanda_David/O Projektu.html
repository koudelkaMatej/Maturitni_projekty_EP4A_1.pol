O projektu

Krátké shrnutí zadání, technologií a toho, jak web a hra spolu mluví.

Cíl

Jednoduchá webová prezentace a účetní vrstva pro hru Snake, aby si hráči mohli vytvářet účty a sledovat nejlepší skóre v jednotlivých módech.

Stack

Backend Flask + SQLite (REST API na http://localhost:5000), frontend statické HTML/CSS/JS servírované přes Live Server, ukládání tokenu v localStorage.

Propojení

Hra i web volají stejné API: /login a /register pro přihlášení, /me pro profil (uživatel, e-mail, datum založení, best skóre), /scores pro ukládání výsledků.

Hlavní funkce

  • Registrace a přihlášení s JWT tokenem, uloženým v localStorage.
  • Stránka Můj účet zobrazuje e-mail, uživatelské jméno, datum založení a nejlepší skóre v jednotlivých módech.
  • Herní mód „Snake" aktualizuje skóre přes API a čte nejlepší hodnoty při startu.
  • Navigace mezi informacemi o hře, módech a účtem je jednotná napříč stránkami.

Jak spustit

  • Spusť backend: python backend/app.py (předpoklad aktivního virtuálního prostředí a závislostí).
  • Frontend otevři přes Live Server z kořene projektu (typicky http://127.0.0.1:5500).
  • V prohlížeči otevři Přihlášení nebo Registrace, vytvoř účet a pokračuj na Můj účet.
