AMP — tvůj fitness deník a průvodce

Aplikace AMP (Advanced Muscle Planner) byla navržena s důrazem na jednoduchost a efektivitu. Cvičení není jen o zvedání vah, ale o disciplíně a sledování pokroku. Naše aplikace vám pomůže udržet si přehled o každém tréninku, každé sérii a každém osobním rekordu.

O projektu (Maturitní práce)

Autor: Student (Maturitní ročník 2026)

Použité technologie:

  • Python 3.12 — Jádro aplikace a logika.
  • Kivy Framework — Grafické uživatelské rozhraní (GUI) pro desktop.
  • Flask — Backendový webový framework a REST API.
  • SQLite 3 — Relační databáze (vztahy 1:N, M:N).
  • HTML5 / CSS3 / JavaScript — Moderní webová prezentace.

Klíčové algoritmy

1. Výpočet osobního rekordu (PR):

SELECT e.name, MAX(wi.weight) 
FROM workout_items wi
JOIN exercises e ON wi.exercise_id = e.id
GROUP BY e.name

2. Kaskádové mazání (Hard Delete): Při smazání uživatele nebo cviku dochází k automatickému odstranění všech souvisejících záznamů v databázi díky ON DELETE CASCADE.

Zdrojový kód

Projekt je spravován v systému Git. Odkaz na repozitář (simulace)

Proč je sledování pokroku klíčové?

Bez systematického záznamu se snadno ztratíte. Zapomenete, jakou váhu jste zvedali minulý týden, kolik opakování jste zvládli, nebo jaký cvik jste chtěli vyzkoušet. AMP řeší tento problém tím, že vám poskytuje nástroje pro:

  • Konzistenci: Pravidelné zaznamenávání vás motivuje k dodržování plánu.
  • Progresivní přetížení: Jasně vidíte, kdy je čas přidat váhu nebo opakování, což je základní princip pro růst svalů a síly.
  • Analýzu a optimalizaci: Díky grafům a statistikám snadno identifikujete, co funguje a kde máte rezervy.
