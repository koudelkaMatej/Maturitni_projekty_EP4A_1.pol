🎯 O projektu

Vize produktu

Space Shooter je plně funkční arkádová hra postavená na Pythonu a Pygame, doplněná o webový backend (Flask + SQLite). Hra není jen technickým cvičením – demonstruje kompletní produkt: od herního enginu přes REST API až po webové rozhraní s žebříčkem.

Proč to zajímá trh

Trh s casual indie hrami roste každý rok. Space Shooter má připravenou základní infrastrukturu pro multiplayer žebříčky, uživatelský management a škálovatelnou obtížnost – klíčové prvky pro udržení hráčů a monetizaci.

🚀 Funkce produktu

🔐

Uživatelský systém

Registrace, přihlášení s hashovaným heslem, host mode pro okamžité hraní.

🏆

Online Leaderboard

Globální žebříček top 10 hráčů s automatickým odesíláním skóre po konci hry.

⚡

Dynamická obtížnost

Nepřátelé se zrychlují každých 10 bodů – přirozená progression curve bez statických levelů.

🎬

Video menu

Animované video pozadí v menu pro profesionální first impression.

🖥️

Web rozhraní

Kompletní webová prezentace hry přístupná z prohlížeče s galerií a žebříčkem.

📐

Responzivní okno

Podpora více rozlišení (800×600 až 1280×960) se zachováním poměru stran.

Python 3 Pygame Flask SQLite REST API Werkzeug Threading pyvidplayer2 HTML/CSS/JS Jinja2
