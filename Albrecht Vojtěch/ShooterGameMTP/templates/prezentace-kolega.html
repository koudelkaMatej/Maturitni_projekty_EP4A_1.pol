Herní engine

Pygame smyčka, sprite systém, vstup hráče, kolize, slow motion, komunikace s API.

Flask backend

REST API pro login, signup, submit_score a scoreboard. Thread-safe SQLite přístup.

Hlavní web stránka

Prezentace hry, galerie screenshotů, lightbox, navigace na registraci.

ScoreBoard.html

Žebříček

Jinja2 template s top 10 hráči z databáze, seřazeno podle max skóre.

database.db

SQLite databáze

Tabulky users a scores s foreign key vazbou.

sprites/ + static/

Assets

Obrázky postav, pozadí, video a fonty použité v Pygame i na webu.

🔌 REST API Endpointy

Metoda Endpoint Popis
POST /api/signup Registrace nového uživatele
POST /api/login Přihlášení, vrací user_id

GET /api/scoreboard Vrátí top 10 hráčů jako JSON
GET /scoreboard HTML stránka žebříčku (Jinja2)

🧩 Klíčové třídy (Pygame)

class Hrac

Hráč

Sprite hráče, pohyb (↑↓), 3 životy, skóre, metoda lose_life() s odesláním skóre.

class Enemy1

Základní nepřítel

Letí zleva, rychlost 3–4 + speed multiplier. Za zásah +1 bod.

class Enemy2

Střílecí nepřítel

Pomalejší, ale generuje EnemyStrela. Za zásah +2 body.

class Strela

Střela hráče

Rychlost 12, spawn z pravé strany hráče. Ničí nepřátele při kolizi.

class EnemyStrela

Střela nepřítele

Rychlost 8 (zpomalena slow motionem). Letí doleva, ničí hráče.

class GameObject

Základní třída

Dědí pg.sprite.Sprite. Načítá sprite z image_path nebo výchozí barevný obdélník.
